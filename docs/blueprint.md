# **App Name**: Doodle Guesser

## Core Features:

- Canvas: Interactive canvas for doodling.
- AI Guess: Use an AI tool to guess the content of the doodle.
- AI Guess Display: Display the AI's guess in an alert box.
- Smiley Background: A background consisting of a mosaic of smiley-face doodles.

## Style Guidelines:

- Primary color: Pale orange (#FFB347) to represent playfulness and warmth.
- Background color: Very light orange (#FFF2E1) to complement the mosaic background, providing a friendly but not distracting base.
- Accent color: Soft yellow (#FCE883) to highlight the interactive button and the alert box for the AI guess.
- Body and headline font: 'PT Sans', sans-serif, will provide a clean, readable look that suits the playfulness of the app, while not feeling childish.
- Use simple, cheerful icons for the doodle and AI functions.
- Add a subtle, playful animation when the AI processes the doodle and provides its guess.
- Center the canvas on the screen with clear buttons for actions below.
